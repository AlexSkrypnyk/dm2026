# Making AI work for you

**Minimising AI slop on Drupal projects** - a talk by Alex Skrypnyk, DrevOps.

AI agents can ship real Drupal features, or they can flood your codebase with slop. The difference is not the model, it is the system you wrap around it. This talk walks through that system: the practices and tools that keep AI-assisted Drupal work clean, tested, and yours.
